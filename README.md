# Mini Dashboard - Estatísticas do Brasileirão 2025

Esse projeto é um site que mostra estatísticas da temporada de 2025 do Campeonato Brasileiro na forma de um pequeno dashboard.

Os dados estão dispostos em forma de gráficos, e apresentam informações como:
- Estatísticas de jogadores
- Estatísticas de goleiros
- Estatísticas de times

O site também inclui a tabela de classificação final do Brasileirão 2025.

O usuário pode utilizar a **barra de navegação** para explorar o conteúdo com mais facilidade.

# Como funciona?

Os dados mostrados em cada um dos gráficos estão armazenados em **arrays em JavaScript**. 

O JavaScript utiliza os dados para gerar os gráficos de **forma dinâmica**.

Durante o desenvolvimento, nós elaboramos uma estrutura prévia para os gráficos (HTML e CSS), e posteriormente implementamos a geração dinâmica por meio do JavaScript, que modifica o **HTML interno** da página.

_**INFORMAÇÃO: GERAÇÃO DE CÓDIGO COM INTELIGÊNCIA ARTIFICIAL:**_
A parte visual da página, bem como estruturação e revisão do código, foi feita completamente por meio de **esforços humanos**, no entanto, contamos com o auxílio de Inteligência Artificial para a geração dos gráficos dinâmicos, e a utilizamos para resolver outras tarefas que seriam repetitivas.

Esse projeto utiliza de logotipos, marcas registradas e dados estatísticos os quais são propriedade da Confederação Brasileira de Futebol (CBF) e dos
respectivos clubes. 
Este projeto tem caráter exclusivamente educacional e não possui qualquer vínculo oficial com a CBF ou com o Campeonato Brasileiro.
